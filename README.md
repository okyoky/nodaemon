# nodaemon - daemon and webserver in node.js

## Installation

```bash
$ npm install
```

### Optionally

For develop:
```bash
$ npm install -g nodemon 
```

For production:
```bash
$ npm install -g forever 
```

## Running
```bash
$ node sample/app 
```
or
```bash
$ ./start sample 
```

It's display this README.md in port 8000
